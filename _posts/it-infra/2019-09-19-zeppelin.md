# 빅데이터 분석을 위한 제플린(zeppelin) 정리

## 제플린 계정 생성 방법

```bash
$ cd zepplin 설치경로 
$ cd zeppelin/conf
$ vim shiro.ini
```

conf/shiro.ini 파일내의 [users] 섹션에 "id = passworld, role" 형식으로 작성. 
비밀번호는 sha-256으로 작성. 
roles은 해당 사용자의 역할이며, [roles] 섹션에서 정의 가능. 

## 제플린 재시작 방법

```bash
$ cd zepplin 설치경로
$ cd ./bin
$ sh zeppelin-daemon.sh restart
```
