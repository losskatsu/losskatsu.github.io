## 2. 장고 이미지 pull 받기 

우선 장고 이미지를 pull받겠습니다. 
다음과 같이 pull을 받아줍시다. 

```bash
$ docker pull django

```

그리고 나서 도커 이미지를 검색하면 django가 있는 것을 알 수 있습니다. 


```bash
$ docker image ls

django        latest    eb40dcf64078   5 years ago     436MB
```
